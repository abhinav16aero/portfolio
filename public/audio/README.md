# Site audio

Drop the menu track here as `menu.mp3` (the MusicPlayer loads `/audio/menu.mp3`).

Notes:
- Use audio you have the rights to. Game OSTs (e.g. Pragmata) are copyrighted —
  host only what you're licensed to use, or a royalty-free track.
- If this file is missing, the music button hides itself automatically.
- Browsers block autoplay with sound until the first user interaction, so the
  track starts on the visitor's first click/scroll (and via the toggle button).
